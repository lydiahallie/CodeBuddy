### Installation
 ```sh
$ git clone git@github.com:lydiahallie/CodeBuddy.git
$ cd CodeBuddy
$ yarn
```
Run client:
```sh
$ cd client 
$ yarn
$ cd ..
$ yarn client
```
Run server:
```sh
$ cd server
$ yarn
$ cd ..
$ ts-node -O '{"module":"commonjs"}' server
```
 ### Branches
Whenever you open a PR, please use the `dev` branch as the base branch.
